-------------------------------------------------- for opencv installation
0. 참고
	https://blog.naver.com/roboholic84/221139256044

1. 다양한 이미지 포맷 지원
	$ sudo apt install -y libjpeg-dev \
						  libtiff5-dev \
						  libjpeg-dev \
						  libpng-dev
					(libjasper-dev 는 ubuntu17 까지만 지원)
2. 다양한 비디오 포맷 지원
	$ sudo apt install -y libavcodec-dev \
						  libavformat-dev \
						  libswscale-dev \
						  libv4l-dev \
						  libxvidcore-dev \
						  libx264-dev

3. OpenCV에는 highgui라는 서브 모듈이 있는데 이미지를 스크린에 출력하거나 간단한 GUI를 만드는데 사용된다. 이에 필요한 라이브러리를 설치 (오래 걸림)
	$ sudo apt install -y libgtk2.0-dev libgtk-3-dev

4. OpenCV 내의 행렬 연산을 최적화시키는데 필요
	$ sudo apt install -y libatlas-base-dev gfortran

5. OpenCV를 파이썬으로 사용할 수 있도록 파이썬 헤더를 설치
	$ sudo apt install -y python2.7-dev
	$ sudo apt install -y python3-dev

6. OpenCV 소스 다운로드
	$ wget -O opencv.zip https://github.com/opencv/opencv/archive/master.zip
	$ unzip opencv.zip
	추가 lib 소스 다운로드
	$ wget -O opencv_contrib.zip https://github.com/opencv/opencv_contrib/archive/master.zip
	$ unzip opencv_contrib.zip
	
#7. PIP를 업데이트
	$ wget https://bootstrap.pypa.io/get-pip.py
	$ sudo python get-pip.py
	$ sudo python3 get-pip.py
	
8. 가상환경에서 빌드 (opencv_contrib 의 경로 설정에 유의)
	$ cd ~/opencv
	$ mkdir build
	$ cd build
	$ cmake -D CMAKE_BUILD_TYPE=RELEASE \
					  -D CMAKE_INSTALL_PREFIX=/usr/local \
					  -D INSTALL_PYTHON_EXAMPLES=ON \
					  -D OPENCV_EXTRA_MODULES_PATH=../../opencv_contrib/modules \
					  -D BUILD_EXAMPLE=ON ..
	$ make
	$ sudo make install
	$ sudo ldconfig

9. OpenCV bindings를 cv 가상환경과 symbolic-link
	$ cd ~/.virtualenvs/cv/lib/python3.x/site-packages/
	$ ln -s /usr/local/lib/python3.x/site-packages/cv2.so cv2.so

10. 확인 (아래는 pip3 install opencv-python 을 설치해서 되는건가??)
	$ python3
	>>> import cv2
	>>> cv2.__version__
