2. Ubuntu 일반설치
	https://rianshin.tistory.com/41
	
		1. Apach       : $ sudo apt install apache2 libapache2 llibapache2-mod-passenger
		2. MySQL      : $ sudo apt install mysql-server mysql-client
		3. Redmine  : $ sudo apt install redmine redmine-sql

		A. /etc/apache2/mods-available/passenger.conf 파일 아래와 같이 편집
			<lfModule mod_passenger.c> PassengerDefaultUser www-data PassengerRoot /usr PassengerRuby/usr/bin/ruby</lfModule>
		B. 링크 생성
			$ sudo ln -s /usr/share/redmine/public /var/www/redmine
		C. default를 수정
			$ sudo nano /etc/apache2/sites-available/default
		D. <Virtualhost>사이의아래를추가한다
			<Directory /var/www/redmine>  RailsBaseURI /redmine  PassengerResolveSymlinksInDocumentRoot on</Directory>
		E. 설정 적용위해 아파치재시작
			$ sudo service apache2 restart
			
		레드마인접속: http://localhost/redmine
